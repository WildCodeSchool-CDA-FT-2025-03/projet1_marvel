# 🏠 Home Center - Votre Bibliothèque Numérique Personnelle

## 📚 À propos du projet

**Home Center** est une application web moderne qui vous permet d'organiser et de découvrir vos médias préférés en un seul endroit. Développée dans le cadre d'un projet de la Wild Code School, cette application vous offre une expérience fluide pour naviguer à travers différentes catégories de médias :

- 📖 Livres
- 🎵 Musique
- 🎮 Jeux Vidéo
- 🎬 Films

Grâce à notre interface intuitive et responsive, vous pouvez facilement explorer notre catalogue, filtrer par catégorie, et découvrir de nouveaux médias pour enrichir votre collection personnelle.

## 🚀 Technologies utilisées

Notre stack technologique a été soigneusement sélectionnée pour offrir les meilleures performances et une expérience de développement optimale :

- ⚛️ **React** - Bibliothèque front-end pour la création d'interfaces utilisateur dynamiques
- 🔷 **TypeScript** - Ajout du typage statique pour un code plus robuste
- 🎨 **TailwindCSS** - Framework CSS utilitaire pour un design responsive et moderne
- 🔄 **GraphQL** - Langage de requête pour API flexible et efficace
- 🗃️ **SQLite** - Système de gestion de base de données léger
- 🔌 **TypeORM** - ORM pour TypeScript et JavaScript
- 🖼️ **Framer Motion** - Bibliothèque d'animations pour des transitions fluides
- 🔍 **Lucide React** - Collection d'icônes minimalistes pour React

## 🤝 Collaboration et contribution

Ce projet est développé par une équipe de 3 personnes dans le cadre de la formation Développeur Concepteur d'Applications à la Wild Code School. Nous utilisons une méthodologie agile pour la gestion du projet avec des sprints hebdomadaires et des stand-ups quotidiens.

### Convention de commits

Nous utilisons la convention de commits conventionnels :

- `feat: description` pour les nouvelles fonctionnalités
- `fix: description` pour les corrections de bugs
- `docs: description` pour les mises à jour de documentation
- `style: description` pour les changements de style
- `refactor: description` pour les refactorisations de code

## 📝 Justification des choix techniques

### Front-end

- **React avec TypeScript** : Permet un développement modulaire avec typage statique pour réduire les bugs.
- **TailwindCSS** : Facilite le développement responsive et maintient une cohérence visuelle.
- **Framer Motion** : Apporte des animations subtiles qui améliorent l'expérience utilisateur.

### Back-end

- **GraphQL** : Offre une flexibilité dans les requêtes et réduit le sur-fetchin/sous-fetching de données.
- **SQLite avec TypeORM** : Solution légère et simple à configurer pour le stockage des données, avec une abstraction ORM puissante.
